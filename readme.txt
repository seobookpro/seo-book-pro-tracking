Add Google Analytics and Google Search Console meta tags to your WordPress site from the admin panel with ease.

== Description ==
The SEO Book Pro Tracking allows you to quickly and easily add:
* Google Analytics tracking code (script) to the footer of your site.
* Google Search Console meta tag to the header of your site.

No need to manually edit your theme files. The SEO Book Pro Tracking makes it easy to configure the important SEO settings from your WordPress admin dashboard.

== Features ==
* Add Google Analytics tracking code via the admin interface.
* Add Google Search Console verification meta tag via the admin interface.
* Automatically injects the Google Analytics code into the footer.
* Automatically injects the Search Console meta tag into the header.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install it directly through the WordPress Plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the **SEO Tracking** menu in your WordPress admin dashboard.
4. Enter your Google Analytics tracking code and Google Search Console meta tag.
5. Save your settings and you're done!

== Frequently Asked Questions ==

= Can I use this plugin with any WordPress theme? =
Yes, the plugin works with any theme and injects the necessary code into the `<head>` and `<footer>` sections of your site.

= Does this plugin support Google Tag Manager? =
Currently, this plugin supports Google Analytics tracking code and Google Search Console meta tags only. Google Tag Manager is not supported in this version.

= How do I verify my site with Google Search Console? =
1. Add the meta tag provided by Google Search Console into the plugin settings under **SEO Tracking**.
2. Save the settings.
3. Go back to Google Search Console and complete the verification process.

= What happens if I deactivate or uninstall the plugin? =
If you deactivate or uninstall the plugin, the Google Analytics and Google Search Console meta tags will no longer appear on your site.
